如何使用 · How to Use

  1. 下載安裝：點上方按鈕下載適合你系統的版本並安裝。
  2. 啟動程式：開啟 MediaGrab，瀏覽器會自動打開操作介面。
  3. 貼上網址：在「智能 Smart」分頁貼上影音網址，按「解析」。
  4. 開始下載：選擇畫質或集數，下載的檔案會存到「下載」資料夾。

常見問題 · FAQ

MediaGrab 是免費的嗎？

是的，完全免費。所有功能都解鎖，沒有付費版。我們透過選用性的廣告與 VPN 推薦連結來支應開發成本，你也可以在設定裡關閉它們。

安裝時出現安全警告怎麼辦？

因為程式未經 Apple/Microsoft 付費簽章，系統會顯示警告。Mac 請到「系統設定 → 隱私權與安全性 → 仍要打開」；Windows 請點「其他資訊 → 仍要執行」。詳見各平台安裝說明。

支援哪些網站？

支援數千個網站，包含 YouTube、Facebook、Instagram、抖音/TikTok、Twitter、Bilibili、Twitch 等。底層使用開源的 yt-dlp 引擎。

下載的內容合法嗎？

MediaGrab 是個人備份工具。請僅用於下載你擁有版權的內容、Creative Commons 授權內容、或你有權保存的公開內容。使用者需自行遵守來源網站的服務條款與當地著作權法。

會收集我的資料嗎？

MediaGrab 本身不收集、不上傳你的下載紀錄或任何個人資料。本下載頁面載入第三方廣告時可能放置 cookie，詳見隱私政策。
